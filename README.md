# Sto based on [Boss](https://github.com/fisherman/boss)

## Install

With [fisherman]

```fish
fisher sto
```

## Features

* Git
    * Dirty
    * Staged
    * Unpushed commits (ahead)
    * Unpulled commits (behind)
    * Unpulled and unpushed commits (diverged)
    * Detached HEAD
    * Branch name
* $status
* $CMD_DURATION
* $status in context
* Background jobs
* Superpowers (sudo)
* Host information
